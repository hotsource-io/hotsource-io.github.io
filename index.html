What you can do

WP Casade lets you hook into any part of WordPress, including the dashboard - making it possible to completely customize your WordPress site to suit your needs.

Here are some examples we've created:

Reward your customers for posting reviews

TRIGGER
When a user posts a review
ACTION
Create a personalized coupon
ACTION
Send the customer a thank you email
Add a custom field to your posts
TRIGGER
When editing a post in the Dashboard
ACTION
Add a Numeric field named "Rating"
ACTION
Add a Text field named "Review Summary"
Add a submission form for your image sharing site
TRIGGER
When a user completes the "Submit Image" form
ACTION
Create an "Image Gallery" post using the form details

TRIGGER
When a user views any page
FILTER
And the user is logged in as a
"Site Member"
ACTION
Display the "[myaccount]" shortcode in the sidebar
